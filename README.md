### Agentic AI Sandbox for Distributed System Design

**SochStack** is an AI-powered educational platform that helps developers learn distributed system architecture through real-time simulation instead of theory. Using a multi-agent system on Amazon Bedrock, it debates architectural decisions, generates Infrastructure-as-Code, deploys mock AWS environments, and visualizes performance bottlenecks using live metrics.
